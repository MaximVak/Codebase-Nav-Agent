# Sample Task Tracker

Sample Task Tracker is a small full-stack JavaScript application used to test Codebase Nav Agent.

The app includes:

- A React frontend
- An Express-style backend
- Authentication routes
- JWT middleware
- A simple database connection file

This sample project is intentionally small so users can quickly test codebase questions such as:

- What does this project do?
- What technologies does this project use?
- Where is authentication handled?
- How does the backend connect to the database?
